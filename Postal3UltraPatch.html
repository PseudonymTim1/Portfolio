Postal III Ultra Patch

About:
Postal III Ultra Patch is pretty much exactly what the title entails. This patch contains many fixes of core issues and crashes with Postal 3 that mostly I fixed while working on the Catharsis Reborn mod, but for Vanilla Postal 3 instead. These make the game much more playable. You won't find these fixes anywhere else, as this patch was made utilizing the source code of the game! Hence the 'Ultra' part of the title. This patch even throws in some bonus cut features to play around with.
