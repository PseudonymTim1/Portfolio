EXCAVATOR

About:
I decided that it'd be fun to recreate Minecraft's procedural terrain generation, as well as other features such as custom biomes and simple block interaction in Unity, all while flexing my optimization skills, this project will probably only go as far as recreating 75% of Minecraft's survival test. If it gets there I'll replace all the Minecraft art and upload a WebGL build so anyone can mess around in it. Might end up as something more than just a minecraft clone for fun. Dunno. Prolly not.
